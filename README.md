# 負荷漸進追蹤（fit-track）

自製的健身**記錄＋報表**一體化 App。純前端 PWA，可「加到主畫面」離線使用，資料只存在自己手機（local-first，無帳號、無伺服器）。

- 線上版：https://nashichuang.github.io/fit-track/
- 安裝：手機瀏覽器開上面網址 → 加到主畫面（Android Chrome／iOS Safari）。

## 功能
- **記錄**：月曆檢視（有訓練的日子打勾），套分部範本一鍵帶出整套動作並帶入上次數據；每組可標正式／熱身、打勾完成。
- **動作庫**：管理動作（綁肌群）與分部範本，依「動作主體→工具」分組排序、可依部位篩選。
- **報表**：單動作（最重組重量／頂組次數／總容量／平均重量／總次數／容量×平均）、肌群（每日總容量／總次數），皆附「過去 N 次/日平均」趨勢線；只計算正式組。
- **備份**：JSON（完整）與 CSV（紀錄）匯出／匯入。

## 技術
- 無框架、原生 ES modules；圖表用 Chart.js（本地版）。
- 資料層 `js/db.js`（IndexedDB）、計算層 `js/metrics.js`、畫面 `js/screens/*`。
- Service Worker (`sw.js`) 負責離線與可安裝；**每次改版需把 `CACHE` 版本號 +1**，使用者重新整理即更新。
